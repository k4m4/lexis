## Install

```
~ ❯❯❯ npm install -g lexis-cli
```


## Usage

```
~ ❯❯❯ lexis --help

	Usage
	  ~ ❯❯❯ lexis <text>
	  ~ ❯❯❯ cat <file> | lexis
	  ~ ❯❯❯ echo <text> | lexis
	Example
	  ~ ❯❯❯ lexis 'Lorem ipsum dolor sit amet'
	  //=> 5
```
